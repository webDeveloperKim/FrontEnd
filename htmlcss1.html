HTML

자세히 보기

HTML(HyperText Markup Language)은 웹페이지를 만드는 데 사용되는 마크업 언어야. 웹사이트의 구조를 정의하는 역할을 하며, 브라우저가 이를 해석해서 화면에 표시하지. HTML의 기본 요소 태그(Tag): HTML은 <태그명> 형태로 구성되며, 대부분의 태그는 <시작태그>내용 형태를 가져. 예)

안녕하세요

→ 큰 제목을 의미함. 속성(Attribute): 태그에 추가적인 정보를 제공하는 요소. 예) Google → href 속성으로 링크를 지정.

CSS

자세히 보기

CSS(Cascading Style Sheets)는 HTML 문서를 꾸미는 데 사용되는 스타일 시트 언어야. HTML이 웹페이지의 구조를 만든다면, CSS는 디자인(색상, 폰트, 레이아웃 등) 을 담당해. CSS의 역할 글꼴, 색상, 크기, 간격 등을 지정 웹페이지의 레이아웃을 조정 (예: 가로 정렬, 배치) 반응형 웹 디자인을 가능하게 함 (모바일, 태블릿 대응) 애니메이션 및 효과 추가

JAVASCRIPT

자세히 보기

JavaScript(JS)는 웹페이지에 동적인 기능을 추가하는 프로그래밍 언어야. HTML이 구조, CSS가 디자인을 담당한다면, JavaScript는 동작(인터랙션, 애니메이션, 데이터 처리 등) 을 담당하지.
